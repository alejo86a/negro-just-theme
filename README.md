# NEGRO JUST - Shopify Theme

A complete, fully responsive, and highly customizable Shopify theme for Atlético Nacional fan gear with a dark, rustic, and passionate aesthetic.

## Theme Overview

The NEGRO JUST theme merges professional e-commerce functionality with a unique, dark rustic design inspired by football culture. The theme features a distinctive "ripped paper" effect, vibrant accent colors, and a passionate verdolaga aesthetic.

## Key Features

### Design & Aesthetics
- **Dark Rustic Theme**: Deep forest green background (#1E442F) with off-white/cream content area (#F0EAD6)
- **Ripped Paper Effect**: Unique torn edge visual effect at top and bottom of main content container
- **Vibrant Accent Colors**: Bright green (#6BFF4E) for CTAs and gold (#FFC700) for premium details
- **Typography**: Bold, all-caps headings (Oswald) with clean body text (Source Sans Pro)

### Functionality
- **Fully Responsive**: Mobile-first design with optimized layouts for all devices
- **Sticky Header**: Navigation stays visible while scrolling
- **Product Card Animations**: Hover effects with image flip/slide animations
- **Quick Add to Cart**: One-click product addition from product cards
- **Mobile Menu**: Clean hamburger menu for mobile navigation
- **Search Integration**: Ready for Shopify's predictive search
- **Newsletter Signup**: Integrated newsletter subscription form
- **SEO Optimized**: Proper heading hierarchy (H1, H2, H3) and meta tags

### Sections Included

1. **Announcement Bar**: Sticky promotional banner at the top
2. **Header**: Main navigation with logo, menu, search, cart, and account icons
3. **Hero Banner**: Large featured banner for featured products/collections
4. **Featured Collection**: Product grid showcasing featured items
5. **Collection Banners**: Split banner layout for category navigation
6. **Featured Product**: Highlight individual products with details
7. **Brands Section**: Showcase related brands/shops
8. **Newsletter & Location**: Two-column layout with address and subscription form
9. **Slogan Section**: Full-width passion statement
10. **Footer**: Structured footer with links and social media

## Installation

1. Upload the theme to your Shopify store via the Theme Editor or Shopify CLI
2. Configure theme settings in the Shopify admin (Customize Theme)
3. Set up collections and products as needed
4. Customize colors, fonts, and content in the theme customizer

## Customization

### Theme Settings

Access theme customization through Shopify Admin > Online Store > Themes > Customize

#### Available Settings:
- **Colors**: Customize primary, accent, and text colors
- **Typography**: Select heading and body fonts
- **Social Media**: Add links to social media profiles
- **Layout**: Adjust page width

#### Section Settings:
Each section can be customized individually:
- Add/remove sections
- Configure content (text, images, collections)
- Adjust color schemes
- Set visibility options
